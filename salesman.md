# 业务员

> 业务员分为 平台业务员/商家业务员/服务商业务员
> 平台 - 业务员 - 服务商
> 服务商 - 业务员 - 商家(二级服务商待定)
> 商家 - 业务员 - 设备

## 新增业务员

| 模块                            | 备注 |
| ------------------------------- | ---- |
| tc.management.management.create |      |


| 参数           | 描述             | 备注 |
| -------------- | ---------------- | ---- |
| type           | 业务员类型         |  0: 平台, 1: 服务商, 2: 商家    |
| relation_id           | 下级id         | 平台业务员为服务商id, 服务商业务员为商家id, 商家业务员为设备id |
