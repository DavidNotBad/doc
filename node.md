## 新增节点提交

| 模块               | 备注 |
| ------------------ | ---- |
| tc.auth.node.store |      |

| 参数   | 描述     | 备注             |
| ------ | -------- | ---------------- |
| name   | 节点名   |       要做限制, 只能是英文字母或下划线           |
| title    | 节点描述   |                  |
| pid    | 父级id   |      顶级节点传递0或''            |
| sort | 排序     |        默认50, 数字越大, 优先级越高          |
| status | 状态     |      '0'=>'关闭', '1'=>'开启'           |
